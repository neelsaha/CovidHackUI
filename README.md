# CovidHackUI

## Steps to RUN:
1) Install NodeJs from NodeJs Official Page.
2) Open Terminal
3) Go to your file project
4) Run in terminal: npm install -g @angular/cli
5) Then: npm install
6) ng serve
7) Navigate to: http://localhost:4200/
